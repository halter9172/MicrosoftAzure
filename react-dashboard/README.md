# Binary Notes Dashboard (React + Vite)

Proyecto base de React/Vite para el dashboard estático.

## Cómo usar

1. Instala dependencias:
   ```bash
   cd react-dashboard
   npm install
   ```

2. Arranca el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construye para producción:
   ```bash
   npm run build
   ```

## Qué incluye

- `src/App.jsx` — layout y renderizado del dashboard
- `src/data.js` — datos iniciales para snippets y métricas
- `src/styles.css` — estilos del dashboard
- `index.html` — punto de entrada para Vite
